This webside provides tour's introduction and booking system for VISA ESG(Environmental Social Governance) Travel.

Calendar

Including tour and hotel booking, I designed two style for the order calendar. One is main for available date, and the other for price. According to the hotel booking rules, if customers book weekend or holiday, they need to pay an additional charge. Cost on calendar is much easier to notice that.

Tour Style - Desktop View
Hotel Style - Mobile View

Furthermore, there has many type of tour's available date. Such as weekend, work day or the first friday of the month. All of those dates aren't stored in database due to limit of the EIP System. Therefore I created a json file to manage that.
